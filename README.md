# Widgey

A lightweight Android "sticky note" widget backed by Workflowy.

Download: <https://github.com/passcod/widgey/releases>

Works offline, both reading and writing.
The only network access needed is when setting up a new widget/note, since we need to fetch the content from Workflowy.

Widgets are resizable to any dimension and can be scrolled.

Workflowy API key: <https://workflowy.com/api-key/>

In the editor, tap on "Synced" to force a resync.

Requires Android 12 at minimum.

---

Build from source with:

```
./gradlew assembleDebug
```
